가상화 window10 64bit에서 설치시 아래와 같이 에러가 발생하여 win7 32bit의 os에서 설치하였다.
  - 에러현상 [ 겁색해서 해결책을 찾아보아도 해결되지 않음]
    - httpd: Syntax error on line 534 of C:/Apache24/conf/httpd.conf: Cannot load c:/php-7/php7apache2_4.dll into server: \xc1\xf6\xc1\xa4\xb5\xc8 \xb8\xf0\xb5\xe2\xc0\xbb \xc3\xa3\xc0\xbb \xbc\xf6 \xbe\xf8\xbd\xc0\xb4\xcf\xb4\xd9.

- php + 아파치서버 설치
	-php설치(ver. 7.1.7)
	  - 다운로드 : http://windows.php.net/download/
	    - 압축을 C:\PHP7에 압축을 푼다.
		- php.ini-development를 복사하여 php.ini파일로 만든다.
		- php.ini파일을 수정한다.
		  - extension_dir = "c:/PHP7/ext"


	-apache설치(ver.2.4.27)
	  - 다움로드 : https://www.apachelounge.com/download/
	    - C:\apache24에 압축을 해제한다.
	    - conf/httpd.conf아래와 같이 추가한다.
		LoadModule php7_module "c:/PHP7/php7apache2_4.dll"
		AddType application/x-httpd-php .php .html
		AddHandler application/x-httpd-php .php .html
		PHPIniDir "c:/PHP7"
	  - cmd상태에서
		C:\Apache24\bin>httpd -t
		Syntax OK

		C:\Apache24\bin>httpd -k install
		Installing the 'Apache2.4' service
		The 'Apache2.4' service is successfully installed.
		Testing httpd.conf....
		Errors reported here must be corrected before the service can be started.
		C:\Apache24\bin>httpd -k start
	  - c:\apache24\htdocs\test.php 파일을 생성한다.
	    - <?php phpinfo(); ?>
